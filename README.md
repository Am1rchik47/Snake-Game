# 🎮 Игра "Змейка" (Snake Game) на Pygame

Динамичная 2D-игра написанная на Python с использованием библиотеки Pygame.

## Примечание
- **Скрипт создает** .txt файл с рекордом в папке где вы его откроете

## 🚀 Механика игры

- **Змейка**: управляется с клавиатуры (клавиши `W`, `A`, `S`, `D`).
- **Цель**: Змейка должна съесть яблоко чтобы набирать очки и бить рекорды

## 🛠 Технологии

- Python 3
- Pygame-ce

## 💻 Как запустить проект

1. **Скачайте** .exe файл в Releases
2. **Запустите** найденный файл

# Или же (для опытных)

1. Установите Pygame-ce (если еще не установлен):
   ```bash
   pip install pygame-ce
   ```
2. Перейдите в папку где находится файл
3. Запустите игру:
   ```bash
   python snake_game.py
   ```
